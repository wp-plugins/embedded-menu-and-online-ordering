== Description ==
A plugin to embed your restaurant menu and ordering into a page or post.

NOTE: This embeds the service provided by Restaurant App Engines into your WordPress site, it does not add menu management to your WorPress site.

Simply change the URL below to match the site URL of your restaurant and copy & paste the following line in your page/post

[EmbedMenu yoursite.restaurantappengines.com 750 2000]  

More Details:

Format:
[EmbedMenu site_url Width Height]  

Example:
[EmbedMenu yoursite.restaurantappengines.com 750 2000]  

Field Definitions:
URL: http://yoursite.restaurantappengines.com/
Width: 750px  
Height: 2000px  


== Installation ==
= How to install Embedded Menu and Online Ordering Plugin (manual) =  
1. Download Embedded Menu Plugin.  
2. Upload it into your plugins directory from Plugin Add Feature or FTP  (this will create a directory /wp-content/plugins/embed-menu-and-online-ordering/)
3. Activate Embedded Menu plugin.
